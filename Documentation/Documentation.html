Edit Portal Settings Module

*** Note ***

This module is only useful if the module DNN-DeleteUnauthorisedUsers has been installed by the host user.

The Edit Portal Settings module is designed to add portal settings to be used by a scheduled task. The Shedule Task is designed to optionally delete unauthorised users after a specified time period since registration. It is an attempt to help administrators in the fight against "spam user registrations". The idea is that the spam users tend to have invalid email addresses and, as a result, never verify their registration and thus do not become authorised. Genuine user registrations will tend to verify their account in the same session.

Four settings are used for each portal:

  1. AutoDeleteUsers
  2. AutoDeleteUsersTimeSpan
  3. AutoPurgeUsers
  4. AutoPurgeUsersTimeSpan

AutoDeleteUsers

This is a true/false value indicating if users should be deleted when the scheduled taks runs.

AutoDeleteUsersTimeSpan

This is a numeric valus indicating the number of minutes to elapse before a user will be deleted id AutoDeleteUsers is set to true.

A deleted user may be restored in DNN by following Users > Manage Users and selecting Deleted to list deleted users. Individual deleted users may be restored. If AutoPurgeUsers is set to true then deleted users can only be restored before they are removed from the database.

AutoPurgeUsers

This is a true/false value indicating if users should be removed from the database when the scheduled taks runs.

AutoPurgeUsersTimeSpan

This is a numeric valus indicating the number of days to elapse before a user will be removed from the database if AutoPurgeUsers is set to true. Once users are removed from the database they cannot be restored.

Usage

The module will take into account the access level of the logged in user. Only administrators or host users can use this module.

Administrators will only see the portal settings for their own portal (website). A host user will see the portal settings for all portals sorted by portal ID.


The Schedule Task is a separate module.
